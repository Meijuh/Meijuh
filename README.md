# Hi, I'm Jeroen Meijer 👋

**Solution Architect** at [Eijsink](https://www.eijsink.nl/) / [DISH](https://www.dish.co/) · **PhD in Computer Science** from the [University of Twente](https://www.utwente.nl/)

I'm a solution architect based in the Netherlands with a background in formal methods, model checking, and automata learning. I hold a PhD from the University of Twente, where I worked on efficient techniques for learning and verifying system behavior. These days I apply that rigour to designing and building enterprise software systems.

---

### 🔧 What I do

- Architect backend systems using **Java**, **Spring**, and cloud-connected infrastructure
- Design real-time distributed messaging with **Redis**, **WebSockets**, and **gRPC**
- Design **advanced algorithms** for optimization and linearization over complex data structures
- Bridge the gap between formal methods research and practical software engineering
- Elevate engineering practices through technical talks, knowledge dissemination, and hands-on mentoring

### 🎓 Academic background

My dissertation, [*Efficient Learning and Analysis of System Behavior*](https://doi.org/10.3990/1.9789036548441) (2019), covers techniques for learning-based testing using active automata learning and LTL model checking, as well as improvements to symbolic reachability analysis. Three of my publications were supported by NASA. I was a core contributor to the [LTSmin](https://github.com/utwente-fmt/ltsmin) model checking toolset and collaborated with the [LearnLib](https://github.com/LearnLib/learnlib) project.

I'm a member of the University of Twente's Industrial Advisory Board (IAB) for the Software Technology Master programme.
